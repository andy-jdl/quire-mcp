## Build Project Arguments
**None**

## Example Prompts
- "Build my quire project day at the park in my Quire Folder."

## Success
- Confirm the Quire project was built by the existance of _site and _epub folders.

## Errors
- **Project does not exist** - confirm project and folder location
- **Timeout** - inform the user the project may still be building and to check their project location
